# 5290 – Probabilidade e Estatística

Este repositório contém o material da unidade curricular (UC) **5290 – Probabilidade e Estatística**, oferecida no Bacharelado Interdisciplinar em Ciência e Tecnologia do Mar (BICT Mar), da Universidade Federal de São Paulo (Unifesp), no $1^o$ semestre de 2026.

Aqui estão organizados o cronograma da UC, as apresentações das aulas, os roteiros de laboratório e outros recursos de apoio ao aprendizado.

## Acesso ao material da UC

Todo o conteúdo da UC pode preferencialmente ser acessado pelo site oficial:

🔗 [https://fcopf.github.io/prob-est-2026](https://fcopf.github.io/prob-est-2026)

O site contém:

- Cronograma das aulas  
- Apresentações utilizadas em sala  
- Roteiros dos laboratórios de informática  
- Plano de ensino  
- Bibliografia e materiais complementares

## Sobre este repositório

Este repositório no GitHub é utilizado para organizar e manter os materiais da UC.

Em geral, **os alunos não precisam utilizar este repositório diretamente**, pois todo o conteúdo relevante está disponível no site oficial.

O repositório existe principalmente para:

- Organizar os materiais da UC
- Manter um histórico das atualizações
- Garantir transparência e acesso público aos conteúdos
